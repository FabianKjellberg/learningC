# ESP32-C6 Learning Project

This repository is used to build a solid foundation in C and ESP-IDF before starting a larger prototype project using the ESP32-C6.

## Purpose

The goal is to understand the language well enough to be able to write correct, structured, and maintainable code for a more complex system later on. Instead of jumping directly into the final project, this is a step to make sure the fundamentals are properly in place.

## What this project contains

The project consists of smaller assignments and experiments, starting from simple syntax and gradually moving toward more involved and structured problems.

The idea is to move from:

* basic C syntax and control flow
* into working with structs and data handling
* and then toward more state-driven and modular designs

before eventually applying that knowledge to a real embedded system.

## Where this is going

The long-term goal is to build a prototype system using the ESP32-C6, focused on plant monitoring and automation.

This will involve working with sensors, collecting and structuring data, and eventually sending that data to a server.

This repository is where the groundwork for that system is being developed.

## Notes

The code in this project is not meant to be final or polished. It is expected to change frequently as new concepts are explored and understood more deeply.
